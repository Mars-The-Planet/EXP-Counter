<center><p style="text-align: center;">This fully configurable client-side mod displays the experience points you've gathered toward your next level, and your total accumulated XP.</p>
<p style="text-align: center;"><img src="https://i.imgur.com/kp9KDkT.png"/></p></center>
<details>
<summary><strong>Configuration</strong></summary>
<p>You can configure this mod directly in-game or edit the JSON file <code>config/expcounter.json</code> in your profile folder. If you're using Fabric and want to adjust the config values in-game, you'll need to install <a href="https://modrinth.com/mod/modmenu">Mod Menu</a>.</p>
<center>
<p style="text-align: center;"><b>show_next_level</b> - Whether the counter of xp collected towards the next level should be displayed</p>
<p style="text-align: center;"><b>show_total</b> - Whether the total accumulated XP should be displayed</p>
<p style="text-align: center;"><b>color_next_level</b> - Color of the counter of xp collected towards the next level should be displayed in decimal</p>
<p style="text-align: center;"><b>color_total</b> - color of the total XP counter in decimal</p>
<p style="text-align: center;"><b>position_off_centre_next_level</b> - X-axis offset off center of the xp to next level counter</p>
<p style="text-align: center;"><b>position_y_next_level</b> - Y-axis position of the xp to next level counter</p>
<p style="text-align: center;"><b>position_off_centre_total</b> - X-axis offset off center of the total xp counter counter</p>
<p style="text-align: center;"><b>position_y_total</b> - Y-axis position of total xp counter</p>
